# have-some-func

This Go application demonstrates the use of CRUD operations on a SQLite database for local data.

## Installation 

Install `go-sqlite3` dependency from source
```
$ go get github.com/mattn/go-sqlite3
```

## Run Project
```
$ go run src/main.go
```
